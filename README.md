# wallpaper project

Projeto de wallpaper interativo estilo HUD, desenvolvido com HTML, CSS e JavaScript.
O objetivo é criar uma interface visual inspirada em sistemas de jogos, exibindo informações em tempo real.

## Funcionalidades

- Relógio em tempo real: Exibição precisa de horas e minutos.

- Data dinâmica: Atualização automática do calendário.

- Integração com API de Clima: Consumo de dados da OpenWeather API para exibir temperatura em tempo real com tratamento de erros.

- Vazio Infinito: Efeito visual de partículas dinâmicas com movimento de profundidade e aceleração via hardware.

- Interface estilo HUD: Layout centralizado com sobreposição de elementos técnicos.

## Tecnologias

- HTML
- CSS
- JavaScript

## Objetivo

Este projeto tem como objetivo praticar desenvolvimento front-end, especificamente o consumo de APIs externas e manipulação assíncrona de dados, criando um wallpaper interativo compatível com ferramentas como Wallpaper Engine.

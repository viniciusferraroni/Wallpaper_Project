# wallpaper project

Projeto de wallpaper interativo estilo HUD, desenvolvido com HTML, CSS e JavaScript.  
O objetivo é criar uma interface visual inspirada em sistemas de jogos, exibindo informações em tempo real.

## Funcionalidades

- Relógio em tempo real
- Data dinâmica
- Interface estilo HUD
- Layout central com sobreposição de elementos

## Tecnologias

- HTML
- CSS
- JavaScript

## Estrutura do Projeto

/assets
/css
/js
index.html

## Objetivo

Este projeto tem como objetivo praticar desenvolvimento front-end e criar um wallpaper interativo compatível com ferramentas como Wallpaper Engine.

## Próximas melhorias

- Integração com API de clima
- Player de música visual
- Animações e efeitos visuais adicionais
